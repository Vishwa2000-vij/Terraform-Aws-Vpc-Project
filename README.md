# Terraform AWS VPC with Public and Private Subnets

## Project Overview

This project provisions a custom AWS VPC using Terraform.

Resources Created:

- VPC
- Public Subnet
- Private Subnet
- Internet Gateway
- NAT Gateway
- Elastic IP
- Public Route Table
- Private Route Table
- Route Table Associations
